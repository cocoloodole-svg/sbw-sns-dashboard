# 信州ブレイブウォリアーズ SNSサマリーダッシュボード

信州ブレイブウォリアーズの公式SNS（X / Instagram / YouTube / Facebook）の投稿実績を集計・可視化するダッシュボードです。

## 公開URL

GitHub Pagesで公開しています。`Settings > Pages` からURLを確認できます。

## 更新方法

現在は週次でスナップショットを手動 / 自動タスクにより再集計し、`index.html` を上書きしてこのリポジトリにプッシュする運用です。将来的にはGitHub Actions + 各種公式API（X API, Meta Graph API, YouTube Data API）へ移行予定です。
